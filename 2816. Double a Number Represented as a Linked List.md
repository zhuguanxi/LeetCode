## Problem Description
You are given the head of a non-empty linked list representing a non-negative integer without leading zeroes.

Return the head of the linked list after doubling it.


### WAY 1:
```
* 使用 stack 需要每次都 new 出新的節點出來，會比直接將 List 反轉做計算更滿，所以可以在優化
class Solution {
public:
    ListNode* doubleIt(ListNode* head) {
        stack<int> s1;

        while (head != nullptr)
        {
            s1.push(head -> val);
            head = head -> next;
        }

        int carry = 0;
        // ListNode* dummy = nullptr;

        while (!s1.empty() || carry) {
            int v1 = 0;
            if (!s1.empty()) 
            {
                v1 = s1.top();
                s1.pop();
            }
            
            int sum = v1 * 2 + carry;

            carry = sum / 10;

            ListNode* node = new ListNode(sum % 10, head);  // 每次 new 會很慢
            head = node;
        }
        return head;
    }
};
```
* n：List 的長度 ([1, 104])
* Time Complexity $O(n)$
* Space Complexity $O(n)$
