# 712. Minimum ASCII Delete Sum for Two Strings

## Problem Link
[Problem](https://leetcode.com/problems/minimum-ascii-delete-sum-for-two-strings/description/?envType=daily-question&envId=2026-01-10)

## Problem Description
Given two strings s1 and s2, return the lowest ASCII sum of deleted characters to make two strings equal.



### WAY 1:
```
#include <vector>
#include <string>
#include <algorithm>
#include <iostream>

using namespace std;

class Solution {
public:
    int minimumDeleteSum(string s1, string s2) {
        int m = s1.size();
        int n = s2.size();
        
        vector<vector<int>> dp(m + 1, vector<int>(n + 1, 0));

        for (int i = 1; i <= m; i++) {
            dp[i][0] = dp[i-1][0] + s1[i-1];
        }
        
        for (int j = 1; j <= n; j++) {
            dp[0][j] = dp[0][j-1] + s2[j-1];
        }

        for (int i = 1; i <= m; i++) 
        {
            for (int j = 1; j <= n; j++) 
            {
                if (s1[i-1] == s2[j-1]) {
                    dp[i][j] = dp[i-1][j-1];
                } else 
                {
                    int delete_s1_char = dp[i-1][j] + s1[i-1];
                    int delete_s2_char = dp[i][j-1] + s2[j-1];
                    dp[i][j] = min(delete_s1_char, delete_s2_char);
                }
            }
        }

        return dp[m][n];
    }
};

```
* s1 and s2 consist of lowercase English letters
* 1 <= s1.length, s2.length <= 1000
* M：s1.length
* N：s2.length
* Time Complexity $O(M) + O(N) + O(M*N) = O(M*N)$
* Space Complexity $O(M*N)$
