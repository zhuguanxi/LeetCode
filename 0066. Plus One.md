# 66. Plus One

## Problem Link
[Problem](https://leetcode.com/problems/plus-one/description/?envType=daily-question&envId=2026-01-01)

## Problem Description
You are given a large integer represented as an integer array digits, where each digits[i] is the ith digit of the integer. The digits are ordered from most significant to least significant in left-to-right order. The large integer does not contain any leading 0's.

Increment the large integer by one and return the resulting array of digits.

### WAY 1:
```
class Solution {
public:
    vector<int> plusOne(vector<int>& digits) {
        
        bool check = false;
        for (int i=digits.size()-1; i>=0; i--)
        {
            if (i==digits.size()-1)
                digits[i] += 1;
            if (digits[i] == 10)
            {
                if (i==0)
                    check = true;
                else
                    digits[i-1]++;
                digits[i] = 0;
            }
        }
        if (!check)
            return digits;
        else
        {
            vector<int> ans(digits.size() + 1, 1);
            std::copy(digits.begin(), digits.end(), ans.begin() + 1);
            return ans;
        }
        
    }
};
```
* N：digits 的長度(1 <= digits.length <= 100)
* Time Complexity $O({2N}) = O({N})$
* 將整個 vector 往後移動就需要 $O({N})$ 成本

### WAY 2:
```
class Solution {
public:
    vector<int> plusOne(vector<int>& digits) {
        int carry = 1;
        for (int i=digits.size()-1; i>=0; i--)
        {
            int tmp_sum = digits[i] += carry;
            digits[i] = tmp_sum % 10;
            carry = tmp_sum / 10;
        }
        if (carry)
            digits.insert(digits.begin(), 1);
        
        return digits;   
    }
};
```
* Time Complexity $O({2N}) = O({N})$
* insert 也是要將整個vector往後移動，所以也需要 $O({N})$ 成本
