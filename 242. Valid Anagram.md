# 242. Valid Anagram

## Problem Link
[Problem](https://leetcode.com/problems/valid-anagram/description/?envType=study-plan-v2&envId=top-interview-150)

## Problem Description
Given two strings s and t, return true if t is an anagram of s, and false otherwise.

### WAY 1:
* Time Complexity O(n), n : s.size()
```
class Solution {
public:
    bool isAnagram(string s, string t) {
        ios_base::sync_with_stdio(0);
        cin.tie(0);
        cout.tie(0);

        if (s.size() != t.size())
            return false;

        map<char, int> ss;
        map<char, int> tt;

        for (int i=0; i<s.size(); i++)
        {
            ss[s[i]]++;
            tt[t[i]]++;
        }

        for (int i=0; i<ss.size(); i++)
        {
            if (ss[i] != tt[i])
                return false;
        }
        return true;
    }
};
```
