# 1351. Count Negative Numbers in a Sorted Matrix

## Problem Link
[Problem](https://leetcode.com/problems/count-negative-numbers-in-a-sorted-matrix/description/?envType=daily-question&envId=2025-12-28)

## Problem Description
Given a m x n matrix grid which is sorted in non-increasing order both row-wise and column-wise, return the number of negative numbers in grid.

### WAY 1:
* Time Complexity O(m*n), grid : m x n matrix
* 當整個矩陣元素皆 > 0 時，時間複雜度會最高
```
class Solution {
public:
    int countNegatives(vector<vector<int>>& grid) {
        int negative_cnt = 0;
        int m = grid.size();
        int n = grid[0].size();

        for (int i=0; i<m; i++)
        {
            for (int j=0; j<n; j++)
            {
                if (grid[i][j] < 0)
                {
                    negative_cnt += (n-j);
                    break;
                }
            }
        }
        return negative_cnt;
    }
};
```
### WAY 2:
* Time Complexity O(m+n), grid : m x n matrix
```
class Solution {
public:
    int countNegatives(vector<vector<int>>& grid) {
        int m = grid.size()
        int n = grid[0].size();
        int i = m - 1, j = 0;

        int res = 0;

        while (i >= 0 && j < n) {
            if (grid[i][j] < 0) {
                res += n - j;
                i--;
            } else {
                j++;
            }
        }

        return res;
    }
};
```
