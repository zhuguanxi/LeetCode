# 238. Product of Array Except Self

## Problem Link
[Problem](https://leetcode.com/problems/product-of-array-except-self/description/?envType=study-plan-v2&envId=top-interview-150)

## Problem Description
Given an integer array nums, return an array answer such that answer[i] is equal to the product of all the elements of nums except nums[i].

The product of any prefix or suffix of nums is guaranteed to fit in a 32-bit integer.

You must write an algorithm that runs in O(n) time and without using the division operation.

### WAY 1:
* Time Complexity O(n*n), n : num.size()
```
class Solution {
public:
    vector<int> productExceptSelf(vector<int>& nums) {
        vector<int>answer(nums.size(),0);
        for (int i=0; i<nums.size(); i++)
        {
            int sum = 1;
            for (int j=0; j<nums.size(); j++)
            {
                if (i != j)
                    sum *= nums[j];
            }
            answer[i]=sum;
        }
        return answer;
    }
};
```
### WAY 2:

* Time Complexity O(n), n : num.size(), Space Complexity O(2n)=O(n)
* 主要是 answer[i] = left[i] * right[i]
```
class Solution {
public:
    vector<int> productExceptSelf(vector<int>& nums) {
        vector<int>answer(nums.size(),1);
        vector<int>left(nums.size(),1);
        vector<int>right(nums.size(),1);

        // O(n)
        for (int i = 0; i < nums.size(); i++)
        {
            if (i != 0)
                left[i] = left[i-1] * nums[i-1];
        }

        // O(n)
        for (int i = nums.size()-1 ; i >= 0; i--)
        {
            if (i != nums.size()-1)
                right[i] = right[i+1] * nums[i+1];
        }

        // O(n)
        for (int i = 0 ; i < nums.size() ; i++)
            answer[i] = left[i] * right[i];

        return answer;
    }
};
```

### way 3:

* Time Complexity O(2n)=o(n), n : num.size(), Space Complexity O(n)
* 從左掃過去在從右邊掃回來
```
class Solution {
public:
    vector<int> productExceptSelf(vector<int>& nums) {
        int n = nums.size();
        vector<int> output(n);
        output[0] = 1;
        for(int i=1; i<n; i++){
            output[i] = output[i-1] * nums[i-1];
        }
        int right = 1;
        for(int i=n-1; i>=0; i--){
            output[i] *= right;
            right *= nums[i];
        }
        return output;
    }
};
```
