## Problem Description
Given a binary array nums and an integer k, return the maximum number of consecutive 1's in the array if you can flip at most k 0's.

### WAY 1:
```
class Solution {
public:
    int longestOnes(vector<int>& nums, int k) {
        int ans = 0;
        int left = 0;
        int n = nums.size();
        int cnt = 0;

        for (int i=0; i<n; i++)
        {
            if (nums[i] == 0)
                cnt ++;
            
            while (cnt > k)
            {
                if (nums[left] == 0)
                    cnt --;
                left ++;
            }

            ans = max(ans, i - left + 1);
        }
        return ans;
    }
};
```
* N：nums 的長度 (1 <= nums.length <= 10e5)
* Time Complexity $O(N)$
* Space Complexity $O(1)$
