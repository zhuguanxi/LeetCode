## Problem Description
Given two integer arrays arr1 and arr2, and the integer d, return the distance value between the two arrays.

The distance value is defined as the number of elements arr1[i] such that there is not any element arr2[j] where |arr1[i]-arr2[j]| <= d.

### WAY 1:
```
class Solution {
public:
    int findTheDistanceValue(vector<int>& A, vector<int>& B, int d) {
        sort(begin(B), end(B));
        int ans = 0;
        for (int n : A) {
            if (upper_bound(begin(B), end(B), n + d) == lower_bound(begin(B), end(B), n - d)) ++ans;
        }
        return ans;
    }
};
```
* N：arr1 的長度 (1 <= arr1.length, arr2.length <= 500)
* M: arr2 的長度
* Time Complexity $O(NlogM)$
* Space Complexity $O(1)$

### WAY 2:
* 只找最接近 n 的那個 B[j]，因為如果最近的都大於 d，其他一定更遠
```
class Solution {
public:
    int findTheDistanceValue(vector<int>& A, vector<int>& B, int d) {
        sort(B.begin(), B.end());
        int ans = 0;

        for (int a : A) {
            auto it = lower_bound(B.begin(), B.end(), a);

            if ((it == B.end() || abs(*it - a) > d) &&
                (it == B.begin() || abs(*prev(it) - a) > d))
                ans++;
        }

        return ans;
    }
};
```
* Time Complexity $O(NlogM)$
* Space Complexity $O(1)$
